## :heavy_check_mark: 소개
     'MOIRA'는 취향기반 관심사 커뮤니티로써, 각각의 카테고리별로 (운동,맛집,스터디,축제) 일회성 모임을 주선하는 플랫폼입니다.

## :heavy_check_mark: 기획의도
    워라밸이 중요한 시대에, 여가및 취미활동에서 관심 분야가 맞는 사람들끼리의 모임을 만들고 싶었습니다. 
    각 관심사별로 다양한 기능을 제공하며 운동,맛집,스터디,축제등의 카테고리로 분류하였습니다.
